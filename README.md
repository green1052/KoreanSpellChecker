# KoreanSpellChecker

코드가 매우 더럽습니다. 풀리퀘좀 해주세요

[한글 맞춤법 검사기](https://speller.cs.pusan.ac.kr/)를 이용한 간단한 파이썬 용 맞춤법 검사기

# 사용법

```py
from spell import spell_check


# text = 검사할 문자
# is_weak = 강한 규칙을 적용할지 여부 기본값 False
spell_check(text, is_weak)
```
